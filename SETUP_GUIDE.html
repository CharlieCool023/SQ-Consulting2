‚ùì Troubleshooting

Q: I got SQL errors

A: That's OK! The SQL is designed to handle errors gracefully. Just run it again.

Q: Still no tables showing in Table Editor

A:

  • Refresh the page
  • Make sure you clicked Run button
  • Try running the SQL again
  • Wait 10 seconds for changes to sync

Q: Diagnostics page still shows ‚ùå

A:

  • Run the SQL again
  • Hard refresh browser (Ctrl+F5)
  • Check .env.local has correct Supabase credentials

Q: Created blog but it doesn't appear

A:

  • Make sure you checked "Published" checkbox
  • Refresh blog page (Ctrl+F5)
  • Check admin dashboard refresh button
